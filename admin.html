Firebase Kurulum Rehberi

  1. Firebase Console'a gidin: console.firebase.google.com
  2. Yeni proje oluşturun veya mevcut projeyi seçin
  3. Web uygulaması ekleyin (Project Settings > Your apps > Web)
  4. Firestore Database oluşturun (Build > Firestore Database)
    Test modunda başlayabilirsiniz
  5. Authentication aktifleştirin (Build > Authentication)
    Email/Password sign-in metodunu etkinleştirin
  6. Admin kullanıcısı oluşturun (Authentication > Users > Add user)
  7. js/firebase-config.js dosyasını Firebase bilgilerinizle güncelleyin
